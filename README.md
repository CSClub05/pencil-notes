# Pencil Notes

A personal note-taking app for iPad. It uses a canvas with Pointer Events for Apple Pencil-friendly drawing and IndexedDB for local saving.

## Features

- Blank canvas note pages
- Pen thickness slider
- Hex color input
- Color picker
- Default black pen
- Apple Pencil, touch, and mouse drawing
- Pressure-aware strokes where supported
- Multiple pages per note
- Multiple note files
- Local autosave with IndexedDB
- Basic offline caching with a service worker
